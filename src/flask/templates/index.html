iNat2wiki is an interface between Wikidata, Wikimedia Commons, Wikipedia, and iNaturalist.

It has 3 core functions:

  • Ordering the contributions by rarity and checking their coverage on Wikipedia and sister projects via "User list"" for users or "Project list"" for iNaturalist projects
  • Adding pictures of particular observations to commons via "Parse observation"
  • Creating Wikipedia stubs (only available for the Wikipedia in Portuguese so far)

It is run and maintained in the context of WikiProject Biodiversity (see the about section for details)
